# Bienvenue dans l'API Limova

L'API Limova est une plateforme complète qui vous permet de créer et gérer des agents IA intelligents, automatiser vos processus de communication et optimiser votre présence en ligne.

## 🚀 Fonctionnalités principales

### Agents IA
- Création et gestion d'agents conversationnels
- Personnalisation des prompts et personnalités
- Gestion des conversations et leads
- Connecteurs pour WhatsApp et autres plateformes

### LIMOVA - Suite d'outils
- **Facturation** : Gestion des paiements et tokens
- **SEO** : Génération automatique d'articles et optimisation
- **Social Post** : Publication automatique sur les réseaux sociaux
- **LinkedIn Automation** : Automatisation complète de LinkedIn
- **Instagram & Facebook** : Gestion des publications
- **Outils** : Génération d'images, reconnaissance vocale, manipulation PDF, présentations

### RobotAppel
- Système d'appels automatisés
- Gestion des notes et numéros de téléphone
- Historique des appels

## 🔑 Authentification

Toutes les requêtes à l'API nécessitent une authentification par token Bearer :

```bash
Authorization: Bearer YOUR_TOKEN
```

## 💰 Système de tokens

L'API utilise un système de tokens pour facturer les services :
- Chaque route a un coût fixe en tokens
- Certaines routes ont un coût variable selon la taille des données
- Consultez la section "Coût" de chaque route pour plus de détails

## 📚 Structure de la documentation

Cette documentation est organisée par modules :
- **Agents** : Gestion des agents IA
- **Conversations** : Gestion des conversations
- **Leads** : Gestion des prospects
- **Connecteurs** : Intégrations avec d'autres plateformes
- **LIMOVA** : Suite d'outils d'automatisation
- **RobotAppel** : Système d'appels automatisés
